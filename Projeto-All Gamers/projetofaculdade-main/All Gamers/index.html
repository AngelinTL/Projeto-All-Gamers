Serviços

  • Game developer

    responsável pela programação e quem dá vida ao jogo a partir do roteiro do Designer. Estuda e aplica as linguagens e ferramentas necessárias para construir o game.

    Acessar

  • Game Designer

    cria a narrativa, escreve o roteiro para o Game Developer programar o jogo. Define a história, o contexto, regras, etapas, desafios, obstáculos, enfim, itens que determinam como o jogador vai passar de fases para vencer o jogo.

    Acessar

  • Game Tester(Qa)

    testa exaustivamente o jogo conforme protocolos e técnicas de teste de nível, caixa preta, jogo isolado, jogo completo etc. Dá aval técnico para liberar o jogo para o mercado. Controla a qualidade do produto.

    Acessar

  • Marketing

    define as estratégias de lançamento do jogo ¿ (promoção, venda, etc.), em diferentes canais do segmento.
